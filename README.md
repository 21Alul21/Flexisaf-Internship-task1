# SimpleBankingApp
This project is the third task for the FlexiSaf Java Backend track. 
This projects simulates a basic banking operations by exposing just four(4) endpoints, to:
- Create a user
- Create an account
- Deposit and
- Withdraw money

## Technologies used
- Java Programming language
- Spring Boot Framework
- JPA
- H2 Database
